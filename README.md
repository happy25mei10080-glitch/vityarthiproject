# Banking System

## Project Title
**Console-Based Banking System**

## Overview of the Project
A simple yet efficient Python-based banking application that simulates real-world banking operations through a console interface. This project demonstrates core banking functionalities including deposits, withdrawals, and balance inquiries, providing users with a seamless banking experience in a command-line environment.

The system is designed to be user-friendly and educational, showcasing fundamental programming concepts while maintaining practical utility for basic banking operations.

## Features
-  Account Verification**: Secure account access using account numbers
-  Deposit Functionality**: Add funds to your account with validation
-  Withdrawal System**: Withdraw money with balance checking
-  Balance Inquiry**: Check current account balance instantly
-  Interactive Menu**: User-friendly text-based navigation
-  Input Validation**: Robust error handling for invalid inputs
-  Secure Exit**: Clean session termination
