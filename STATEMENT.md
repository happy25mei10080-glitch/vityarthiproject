# Project Statement Document

## Problem Statement

Many people struggle with managing their basic banking transactions and keeping track of their account balances.  
There is a need for a simple tool where users can practice and understand basic banking operations like deposits, withdrawals, and balance checks without the complexity of real banking systems.

## Scope of the Project

This project focuses on building a basic command-line banking application that helps the user:
- Verify bank account access
- Deposit money into their account
- Withdraw money with balance validation
- Check current account balance
The scope is limited to simple transaction handling using Python variables and does not include advanced features like permanent data storage, transaction history, or database integration.

## Target Users

The project is mainly meant for:
- Programming students learning Python
- Banking trainees understanding basic operations
- Individuals practicing financial literacy
- Anyone who wants to understand basic banking transactions

## High-Level Features
- Account verification using account numbers
- Deposit money with amount validation
- Withdraw money with balance checking
- Display current account balance
- Simple text-based menu for easy navigation
